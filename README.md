Hotel Chatbot

Αυτό το project είναι ένας έξυπνος βοηθός (chatbot) για το "The Grand Resort Hotel". Χρησιμοποιεί Τεχνητή Νοημοσύνη (LLMs), τεχνικές RAG (Retrieval-Augmented Generation) και εργαλεία αναζήτησης σε βάση δεδομένων για να παρέχει πληροφορίες και να εξυπηρετεί τους πελάτες του ξενοδοχείου.

Χαρακτηριστικά

- Συχνές Ερωτήσεις (FAQ): Απαντήσεις σε κοινές απορίες μέσω διανυσματικής αναζήτησης (FAISS).
- Κριτικές Πελατών: Περιλήψεις και αναζήτηση κριτικών για διάφορες υπηρεσίες του ξενοδοχείου.
- Σημεία Ενδιαφέροντος (POIs): Πληροφορίες για αξιοθέατα και τοποθεσίες κοντά στο ξενοδοχείο.
- Κρατήσεις & Δωμάτια: Έλεγχος διαθεσιμότητας και πληροφορίες για τύπους δωματίων μέσω SQL βάσης δεδομένων.
- Πολυγλωσσική Υποστήριξη: Διεπαφή σε Ελληνικά και Αγγλικά.
- Σύστημα Feedback: Δυνατότητα αξιολόγησης των απαντήσεων για συνεχή βελτίωση.

Προαπαιτούμενα

- Python: Έκδοση 3.9 ή νεότερη.
- DeepSeek API Key: Απαιτείται για τη λειτουργία του μοντέλου γλώσσας. Μπορείτε να το αποκτήσετε από το [DeepSeek Platform](https://platform.deepseek.com/).

Εγκατάσταση

Ακολουθήστε τα παρακάτω βήματα για να στήσετε το project τοπικά:

1. Προετοιμασία Περιβάλλοντος

Δημιουργήστε ένα virtual environment και ενεργοποιήστε το:

```bash
# Δημιουργία
python -m venv .venv

# Ενεργοποίηση (Mac/Linux)
source .venv/bin/activate

# Ενεργοποίηση (Windows)
.venv\Scripts\activate
```

2. Εγκατάσταση Εξαρτήσεων

Εγκαταστήστε τις απαραίτητες βιβλιοθήκες:

```bash
pip install -r requirements.txt
```

3. Ρύθμιση Μεταβλητών Περιβάλλοντος

Δημιουργήστε ένα αρχείο με όνομα `.env` στον κεντρικό φάκελο του project και προσθέστε το API Key σας:

```env
DEEPSEEK_API_KEY=το_κλειδι_σας_εδω
```

Αρχικοποίηση Δεδομένων (Προαιρετικά)

Η βάση δεδομένων `hotel.db` περιλαμβάνεται ήδη στο project με προ-εγκατεστημένα δεδομένα (δωμάτια, κρατήσεις κλπ.). Επίσης, τα διανυσματικά ευρετήρια (vector stores) υπάρχουν ήδη στον φάκελο `vector_stores/`.

Αν παρ' όλα αυτά θέλετε να κάνετε reset ή να αρχικοποιήσετε τα πάντα από την αρχή:

1. Δημιουργία SQL Βάσης Δεδομένων:
   ```bash
   python setup_db.py
   python setup_feedback_db.py
   ```

2. Δημιουργία Vector Stores:
   ```bash
   python setup_vector_stores.py
   ```

Εκτέλεση της Εφαρμογής

Για να ξεκινήσετε το chatbot με το περιβάλλον εργασίας Streamlit, τρέξτε:

```bash
streamlit run app.py
```

Η εφαρμογή θα είναι διαθέσιμη στη διεύθυνση `http://localhost:8501`.

Επιπλέον Λειτουργίες

Βελτίωση μέσω Feedback
Αν θέλετε να ενσωματώσετε τις "καλές" απαντήσεις (αυτές που έλαβαν θετικό feedback) στη μνήμη του chatbot για μελλοντική χρήση:
```bash
python update_good_conversations_vector_store.py
```

Αξιολόγηση (Evaluation)
Για να εκτελέσετε τα σενάρια ελέγχου και να δείτε την απόδοση του chatbot:
```bash
python evaluation.py
```

Δομή Project

- `app.py`: Η κύρια εφαρμογή περιβάλλοντος χρήστη (Streamlit).
- `chatbot_logic.py`: Η κεντρική λογική, τα tools και το LangChain agent.
- `Datasets/`: Τα αρχικά δεδομένα σε μορφή CSV (FAQ, Reviews, POIs).
- `vector_stores/`: Αποθηκευμένα ευρετήρια FAISS.
- `hotel.db`: SQLite βάση δεδομένων για δωμάτια και feedback.
